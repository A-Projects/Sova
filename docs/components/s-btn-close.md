# Кнопка закрытия

Компонент `s-btn-close` представляет из себя лишь кнопку закрытия. 

### Как использовать
Т1

<s-btn-close/>

``` vue
<s-btn-close/>
```

### Отключена
Т2

<s-btn-close disabled/>

``` vue
<s-btn-close disabled/>
```

### Белая
Т3

<div class="bg-secondary d-flex align-items-center" style="height: 40px;">
<s-btn-close white/>
<s-btn-close white disabled/>
</div>

``` vue
<s-btn-close white/>
<s-btn-close white disabled/>
```
